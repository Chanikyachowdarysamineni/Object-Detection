# 🎯 Real-Time Object Detection with YOLOv8

A complete full-stack application for real-time object detection using YOLOv8, FastAPI, PostgreSQL, and Gradio.

## 🎯 Features

- **Real-Time Detection**: Detect objects from images, videos, and webcam streams
- **YOLOv8 Integration**: High-performance object detection model
- **Web UI**: Interactive Gradio interface for easy use
- **REST API**: FastAPI backend for programmatic access
- **Database Logging**: PostgreSQL for storing all detection results
- **Statistics & Analytics**: Track detection history and class distribution
- **Docker Support**: Containerized deployment ready
- **Model Fine-Tuning**: Ready for training on custom datasets
- **Production Ready**: Authentication, CORS, error handling, logging
